#### `detect_backend_env_vars` priority logic

1. `${VAR}` expansion → `__UAT_SYNTHETIC__` placeholder (per §4 ZAKÁZANÉ čítať host env)
2. `DB_CONNECTION_VARS` explicit whitelist → rewrite na UAT container hostname + detected/synthetic creds
3. SECRET_SUFFIXES (`_password/_secret/_key/_token` case-insensitive) → `secrets.token_hex(32)` random
4. Plain values → copy as-is

#### Conditional Step 8 alembic (CR-022c)

- `self-bootstrap` → skip external call (backend lifespan handler runs migrations)
- `external` → `python -m alembic upgrade head` first, fallback `poetry run alembic`
- `skip` → no call

#### 4 nové CLI flags

`--alembic-strategy {auto,self-bootstrap,external,skip}`, `--skip-env-detection`, `--db-user`, `--db-name`.
